Hello!

Srhuman's Balatro Mod Loader is a mod loader made for the game Balatro by LocalThunk.

It is injected using lovely, an injector for the game framework LOVE2D which Balatro is built on.

You can find lovely here

Wiki / Documentation?

There won't be a Documentation or Wiki, this mod loader only comes with a few methods to make modding easy, I will make some API mods to extend on this

As of right now you are unable to make a now game object / item, without a mod installed that adds features to that
